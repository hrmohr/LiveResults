In order to use this you will have to configure the following options:

competition.id
password
results.filename

This can be done directly in the conf/config.properties file or by using the built in preferences dialog.

* Special notes for running on Microsoft Windows *

You might have to enter the path using double backslashes. Example: C:\\path\\to\\spreadsheet.xls
